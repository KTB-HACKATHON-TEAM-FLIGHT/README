## 👋 About Project 
`ChatPPT` 는 AI 기반의 대화형 인터페이스를 통해 사용자가 원하는 주제, 스타일, 슬라이드 구성 등을 간단한 대화로 요청하면 AI가 자동으로 PPT를 생성하는 서비스 입니다.

## ⭐Service and Function
복잡한 도구 사용    ➡️  간단한 대화만으로 PPT 제작!
